USO DE LAS ETIQUETAS SEMÁNTICAS, LISTAS Y ENLACES

Este sitio web se lo crea con el propósito de aprender acerca
del uso e implementación de las etiquetas semánticas, listas
y enlaces en el diseño de páginas web

BORRADOR DE CONTENIDO

Para comprender mejor las temáticas que se van a estudiar en
la presente página se hará una breve definición de las mismas.

Etiquetas semánticas

Se trata de un tipo de etiqueta llamada semántica porque da
noción del tipo de contenido que englobará. Están pensadas
para que la estructura de la página web sea más lógica y
funcional (Celaya, 2014).

Algunas de ellas implementadas en HTML5 son:

  • <header>... </header>: Para definir la cabecera de la página.


  • <footer>...</footer>: Para definir el pie de página.
    Normalmente contiene datos sobre quien ha creado la página, datos del
    copyright, etc.


  • <nav>...</nav>: Permite definir un elemento de navegación de
    la página web, como el menú principal o menús secundarios.


  • <section>...</section>: Se utilizan para encerrar el código
    correspondiente a una sección genérica dentro de un documento.


  • <aside>...</aside>: Sirve para delimitar todo aquel contenido
    que no es directamente contenido principal de esa página en concreto. Puede
    usarse por tanto para todos aquellos elementos secundarios como bloques
    publicitarios, enlaces externos, citas, un calendario, etc.


  • <article>...</article>: Se utiliza para definir artículos como
    noticias, entradas de un blog, etc. Es probablemente uno de los más
    importantes de HTML5, ya que permite indicar cuál es el contenido de una
    página web (Celaya, 2014).

    • Listas y enlaces

    Como lo indica su nombre las listas permiten ordenar la información
    en listas, es decir que la información va una debajo de la otra.

    Existen dos tipos de listas:

    Listas no numeradas: se emplean cuando no se tiene en cuenta el orden,
    se representan por la etiqueta <ul> ... </ul>.

    Listas numeradas: se emplean cuando se tiene en cuenta el orden,
    se representan por la etiqueta <ol> ... </ol>.

    Dentro de estas etiquetas se coloca la etiqueta <li> ... </li>
    la cual representa cada elemento dentro de la lista.

    Los enlaces son direcciones que permite comunicarse con otra página
    o con contenido de la misma página. Para ingresar un enlace se hace
    uso de la etiqueta <a href="" > </a>.

    Para profundizar mas acerca de estos temas, diríjase a la sección de
    de temas, allí encontrará los tipos, la utilidad y la importancia de
    cada tema en estudio.
