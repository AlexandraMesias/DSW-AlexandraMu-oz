Listas y Enlaces

Listas

Como lo indica su nombre las listas permiten ordenar la información en listas, es decir que la información va una debajo de la otra.

Existen tres tipos de listas:

Listas no numeradas

Se emplean cuando no se tiene en cuenta el orden, se representan por la etiqueta <ul> ... </ul>.


Etiqueta ul. Obtenido de: https://4.bp.blogspot.com/

Listas numeradas

Se emplean cuando se tiene en cuenta el orden, se representan por la etiqueta <ol> ... </ol>.


Etiqueta ol. Obtenido de: https://1.bp.blogspot.com/

Dentro de las etiquetas para las listas no numeradas y las listas numeradas se coloca la etiqueta <li> ... </li> la cual representa cada elemento dentro de la lista, como se ha visto en las imágenes anteriores.

Listas de definiciones

Se utilizan cuando se quiere realizar una lista de definiciones, se representa mediante las siguientes etiquetas.

<dl> ... </dl> Se utiliza para indicar que se colocará una definición. Dentro de esta etiqueta van las siguientes dos etiquetas.

<dt> ... </dt> Dentro de esta etiqueta va el término que se va a definir, es decir el título.

<dd> ... </dd> Dentro de esta etiqueta se coloca la definición del término.

Enlaces

Los enlaces son direcciones que permite comunicarse con otra página o con contenido de la misma página. Para ingresar un enlace se hace uso de la etiqueta <a href="" > </a>.
